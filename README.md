# Лапки і Серця - Притулок для тварин
## Опис проекту
Знайди собі друга:
Переглянь наших тварин, які шукають свої нові домівки. Кожна тварина пройшла медичний огляд, отримала необхідні щеплення та готова до адопції.
## Структура проекту
Проект організовано згідно стандартної структури:
/ Lab2_project
    /images – папка, що містить зображення сайту (логотип, фото тварин)
    /css/
        base.css - базвоі стилі тексту, заголовків, посилань, абзаців
        components.css - стилі для навігаційного меню, гамбургер-меню, форм, міток, кнопок, та інших компонентів
        layout.css - структурні елементи сторінки, налаштування медіа запитів для адаптивного дизайну
        style.css - головний файл, що імпортує всі складові
    /js
        script.js -  шматочок коду на Java-скрипті для роботи бургер-меню
about.html - Про нас, історія створення притулку
adoption.html – Сторінка з переліком та описом тварин, доступних для адаптації
contacts.html - Зв’яжіться з нами, сторінка з контактною інформаціє та форма зворотного зв’язку
help.html - Сторінка з інформацією для бажаючих допомогти
index.html -  Головна сторінка сайту, містить короткий опис притулку

## Короткий опис застосованих технологій (Flexbox, медіазапити).
1) Використано flexbox для побудови адаптивного макету, а саме: 
-у навігаційному меню (nav ul { display: flex; justify-content: center;})
-для вирівнювання елементів у заголовці (header { display: flex; justify-content: space-between; /*логотип ліва, гамбургер-меню - справа*/align-items: center;})

2) Використано медіа запити (@media):
-для відображення у мобільній версії бургер-меню замість звичайного (розділ @media (max-width:768px))

3) адаптивні зображення (main img {max-width: 100%;height: auto;})

4) Базова клієнтська валідація форм:
HTML5-атрибути: required, minlength, maxlength, pattern, title 
Стилі для відображення помилок:
input:invalid - червона підсвітка для полів, що заповнені з помилками
input:valid - зелена підсвітка для коректно заповнених полів
5) Елементи JavaScript:
Додано обробку подій - при кліці на кнопку до елементу hamburger-menu та nav-menu
додається або прибирається класс active для того щоб прибирати або додавати відповідні стилі
